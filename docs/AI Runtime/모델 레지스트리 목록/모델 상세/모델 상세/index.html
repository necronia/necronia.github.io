모델 상세

모델 레지스트리 목록에 나오는 모델을 선택했을 때 보여지는 화면이다.

모델 상세 화면#

1. 공통 GNB#

  • 공통 GNB 부분입니다.
    • 대시보드 - 공통 GNB 화면으로 이동합니다.

2. 모델 정보#

  • 모델의 정보를 보여줍니다.
    • 모델명
    • 모델상태 : RUNNIMG(녹색), WARNING(주황색), ERROR(빨간색)
    • 상태 업데이트 시간/ 업데이트 작업자
    • 생성일 / 생성자

3. 모델해석 요청#

  • 모델 데이터 타입이 Tabular 인 경우 해석요청
    • Tabular 데이터 타입에 대한 모델해석을 요청 할 수 있습니다.
  1. 해석에 필요한 자원요청
    • 해석에 필요한 자원을 설정합니다.
  2. 모델 데이터 타입
    • 모델 등록 시 설정된 데이터 타입(Tabular)을 보여줍니다.
  3. 해석 Tabular 데이터 등록
    • 해석 요청 할 Tabular 데이터를 최대 5개까지 등록합 수 있습니다.
  • 모델 데이터 타입이 Text 인 경우 해석요청
    • Text 데이터 타입에 대한 모델해석을 요청 할 수 있습니다.
  1. 해석에 필요한 자원요청
    • 해석에 필요한 자원을 설정합니다.
  2. 모델 데이터 타입
    • 모델 등록 시 설정된 데이터 타입(Text)을 보여줍니다.
  3. 해석 Text 데이터 등록
    • 해석 요청 할 Text 데이터를 최대 5개까지 등록합 수 있습니다.
  • 모델 데이터 타입이 이미지 인 경우 해석요청
    • 이미지 데이터 타입에 대한 모델해석을 요청 할 수 있습니다.
  1. 해석에 필요한 자원요청
    • 해석에 필요한 자원을 설정합니다.
  2. 모델 데이터 타입
    • 모델 등록 시 설정된 데이터 타입(이미지)을 보여줍니다.
  3. 해석 이미지 데이터 등록
    • 해석 요청 할 이미지 데이터를 최대 5개까지 등록합 수 있습니다.
  • 모델 해석 요청 후 확인
    • 해석이력에서 요청한 데이터 개수만큼 모델 해석이 진행 중 인 것을 확인 할 수 있습니다.
    • 모델 해석 중 일 때는 모델해석을 요청 할 수 없습니다.

4. 컨텍스트 메뉴#

  • 컨텍스트 메뉴를 이용하여 아래 동작을 수행할 수 있습니다.

  • 모델배포 자원 정보

    • 배포자원 생성이 되지 않을 때 선택을 하면 아래와 같은 안내 팝업이 나옵니다.
    • 배포자원 생성은 모델 레지스트리 목록 - 3. 모델 배포자원 참조하시면 됩니다. [모델 레지스트리 목록]

  • Access Key 관리

    • 모델 예측을 위한 Access Key 발급과 관리를 할 수 있는 기능입니다.

  • 모델 삭제

    • 모델 삭제를 수행합니다. 서빙 중 일 때는 삭제가 되지 않습니다.

5. 모델 상세 개별 탭 화면#

  • 버전 현황
    • 버전 현황 화면으로 이동합니다.
  • 서비스 대시보드
    • 서비스 대시보드 화면으로 이동합니다.
  • 예측 이력
    • 예측 이력 화면으로 이동합니다.
  • 해석 이력
    • 해석 이력 화면으로 이동합니다.
  • 모델 Drift
    • 모델 Drift 화면으로 이동합니다.
