모델 레지스트리 목록

모델 레지스트리에 속해 있는 모델의 상태를 보여주는 기능을 제공하고 있습니다. 모델 자원배포 생성, 수정 및 모델 등록도 수행을 합니다.

모델 레지스터리 화면#

1. 공통 GNB#

  • 공통 GNB 부분입니다.
    • 대시보드 - 공통 GNB 화면으로 이동합니다.

2. 모델 레지스트리 UI 선택#

  • 카드 목록 과 테이블 중 선택할 수 있는 기능입니다.

3. 모델 리스트 소팅 기능#

  • Owner, 모든 모델/서빙 중 모델, 최신 업데이트 순/제목 오름차순/제목 내림차순, 문자열 검색으로 모델을 검색하거나 필터링 하는 기능입니다.

4. 검색창#

  • 모델명 문자열 검색으로 모델을 검색하는 기능입니다.

5. 모델 배포자원#

  • 모델 레지스트리에 배포자원이 생성되어 있으면 배포자원 정보로 표시되고 생성된 배포자원이 없으면 배포자원 생성으로 배포자원을 생성할 수 있습니다.

  • Small, Medium, Large, 직접설정으로 배포자원을 설정할 수 있습니다.

  • 생성된 배포자원은 수정기능을 이용하여 수정할 수 있습니다.

6. 모델 등록#

  • 버전 1 의 새로운 모델을 등록할 수 있는 기능입니다.

  • 모델명 중복 체크를 해야 Weight/사용자 코드/학습 데이터를 설정할 수 있습니다. Weight/사용자 코드는 필수항목 입니다.

  • 모델명 중복체크 후 프레임워크/Weight/사용자 코드/학습 데이터를 설정하면 모델을 등록할 수 있습니다.

  • 등록된 모델은 모델 레지스트리에서 확인 할 수 있습니다.

7. 모델 정보#

  • 모델 레지스트리 명, 모델명, 모델상태(HEALTHY, WARNING, ERROR)를 보여줍니다.

8. 모델 버전 추가/삭제#

  • 모델 버전 추가나 삭제를 실행합니다. 서빙 중 일때는 삭제할 수 없습니다.
