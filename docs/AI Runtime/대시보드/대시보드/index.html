대시보드

대시보드는 모델 레지스트리의 모델 현황 및 자원 상태를 확인 할 수 있는 기능을 제공합니다.
이 기능 외에 사용자 계정 관리, 로그아웃 및 프로젝트 생성 기능도 있습니다.

대시보드 화면#

1. 모델 레지스터리 현황#

  • 클러스터 이름으로 자원을 검색합니다.모델 레지스터리의 상태를 파악 할 수 있는 화면을 제공하며 해당 항목의 숫자를 클릭하면 선택한 상태의 모델만 보여주게 됩니다.

2. 자원관리 현황#

  • 클러스터 와 네임스페이스의 자원 상태를 파악 할 수 있는 기능입니다.

3. 보기 모드#

  • 모델 레지스트리 목록은 보기 모드 선택에 따라 카드와 테이블 형태로 보여질 수 있습니다.

4. 모델 레지스터리 목록#

  • 모델이 존재하는 모델 레지스트리들만 표시가 됩니다.

  • 모델 레지스트리 이름을 선택하면 모델 레지스트리 목록 화면으로 이동하고 모델을 선택하시면 모델 상세로 이동됩니다.

  • 메뉴를 선택하여 모델버전 추가 화면으로 이동이나 삭제를 하실 수도 있습니다. 서빙 중에는 삭제가 안됩니다.

5. 공통 GNB#

  • AccuInsight 및 대시보드 이동
    • 공통 GNB가 보이는 화면에서 AccuInsight 아이콘을 선택하면 AccuInsight 으로 이동하고 AI Runtime을 선택하면 대시보드 화면으로 이동됩니다.
  • 사용자 계정 관리 및 로그아웃
    • 사용자 이름이나 아이콘을 선택하면 사용자 계정 정보를 수정할 수 있는 팝업이 나타나고 사용자가 admin 계정이면 Admin 콘솔로 이동할 수 있는 항목도 보여지게 되고 로그아웃도 수행할 수 있습니다.

6. 프로젝트 생성#

  • AI Runtime 단독모드로 실행을 하면 아래처럼 프로젝트 생성 버튼이 보이고 클러스터 및 자원을 지정하여 프로젝트를 생성할 수 있습니다. 통합모드에서는 보여지지 않습니다.
