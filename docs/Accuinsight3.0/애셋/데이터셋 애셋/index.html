데이터셋 애셋

전사 레벨로 공유된 데이터셋 애셋을 조회하고 관리합니다.

데이터셋 애셋 화면#

1. 보기 모드#

카드 뷰와 리스트 뷰를 선택할 수 있습니다.

  • 데이터셋 애셋 카드

  • 데이터셋 애셋 리스트

2. 산업군 필터#

선택한 산업군의 애셋만 표시합니다.

  • 산업군 목록
    • Commerce
    • Construction
    • Distribution
    • Energy
    • Environment
    • Finace
    • Healthcare
    • Housing
    • ICT
    • Life
    • Manufacturing
    • Mobility
    • None
    • Public
    • Trade
    • Vision

3. 정렬#

  • 추천순 : 프로페셔널 애셋이 우선 정렬되며, 나머지는 최신 업데이트 순으로 정렬 됩니다. 데이터셋 애셋은 프로페셔널 애셋이 없기 때문에 추천순을 선택하더라도 최신 업데이트 순과 동일하게 정렬 됩니다.
  • 최신 업데이트 순 : 가장 최근에 업데이트된 애셋 순으로 정렬 됩니다.
  • 오래된 순 : 업데이트가 오래된 애셋 순으로 정렬 됩니다.

4. 검색#

검색어를 입력 하면 애셋 이름, 프로젝트 이름, 공유자 이름을 대상으로 검색된 결과를 표시합니다.

검색어를 입력하면 결과가 검색창 하단에 미리 보여지며 엔터를 입력하면 애셋 목록에 반영됩니다.

More Actions#

카드 뷰 More Actions#

카드에 마우스 커서를 올리면 more actions 버튼이 표시되며, 데이터셋 애셋은 다음과 같은 메뉴를 제공합니다.

  • 새 워크플로우 시작 : 데이터셋 애셋을 사용하여 새로운 워크플로우를 생성 합니다.
  • 삭제 : 데이터셋 애셋을 삭제 합니다. 삭제 권한이 있을 때에만 표시 됩니다.

리스트 뷰 More Actions#

리스트의 more actions 버튼은 항상 표시되며, 카드 뷰와 동일한 메뉴를 제공 합니다.

다중 선택 More Actions#

여러개의 항목을 선택해서 More Actions를 실행할 수 있는 기능을 제공 합니다. 워크플로우를 생성할 때 여러개의 데이터셋 애셋을 사용 가능 합니다.

  • 카드 뷰 다중선택

    카드에 마우스 커서를 올리면 선택 체크박스가 표시됩니다. 삭제와 워크플로우 시작 버튼이 표시 됩니다.

  • 리스트 뷰 다중선택

    리스트의 선택 체크박스는 항상 표시되며, 제공되는 액션 버튼은 카드 뷰와 동일 합니다. 삭제와 워크플로우 시작 버튼이 표시 됩니다.

워크플로우 시작#

데이터셋 애셋을 사용해 새로운 워크플로우를 생성합니다.

More Actions에서 워크플로우 시작 버튼을 선택하면 다음과 같은 프로젝트 선택 팝업이 표시 됩니다.

프로젝트를 선택하면 선택한 프로젝트에 새로운 워크플로우가 생성 됩니다. 새로 생성된 워크플로우에는 데이터셋 애셋에 저장되어 있던 데이터셋이 포함됩니다.
