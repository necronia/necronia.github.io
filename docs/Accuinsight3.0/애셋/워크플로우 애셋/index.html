워크플로우 애셋

전사 레벨로 공유된 워크플로우 애셋을 조회하고 관리합니다.

워크플로우 애셋 화면#

1. 보기 모드#

카드 뷰와 리스트 뷰를 선택할 수 있습니다.

  • 워크플로우 애셋 카드

  • 워크플로우 애셋 리스트

  • 애셋 구성요소 표시

    • : 워크플로우 애셋에 리포트가 있음
    • : 워크플로우 애셋에 샌드박스가 있음
    • : 워크플로우 애셋에 모델이 있음

2. 프로페셔널 애셋 필터#

프로페셔널 애셋은 공식적으로 기본 제공하는 애셋입니다. 필터 선택 시 프로페셔널 애셋과 같은 그룹의 사용자가 전체 공개한 애셋만 표시 합니다.

3. 산업군 필터#

선택한 산업군의 애셋만 표시합니다.

  • 산업군 목록
    • Commerce
    • Construction
    • Distribution
    • Energy
    • Environment
    • Finace
    • Healthcare
    • Housing
    • ICT
    • Life
    • Manufacturing
    • Mobility
    • None
    • Public
    • Trade
    • Vision

4. 애셋 구성요소 필터#

선택한 구성요소를 포함한 애셋만 표시합니다.

선택 가능한 항목은 "전체", "리포트", "Sandbox", "모델"이며 "전체"를 제외한 나머지는 다중 선택이 가능합니다.

5. 정렬#

  • 추천순 : 프로페셔널 애셋이 우선 정렬되며, 나머지는 최신 업데이트 순으로 정렬 됩니다.
  • 최신 업데이트 순 : 가장 최근에 업데이트된 애셋 순으로 정렬 됩니다.
  • 오래된 순 : 업데이트가 오래된 애셋 순으로 정렬 됩니다.

6. 검색#

검색어를 입력 하면 애셋 이름, 프로젝트 이름, 공유자 이름을 대상으로 검색된 결과를 표시합니다.

검색어를 입력하면 결과가 검색창 하단에 미리 보여지며 엔터를 입력하면 애셋 목록에 반영됩니다.

More Actions#

카드 뷰 More Actions#

카드에 마우스 커서를 올리면 more actions 버튼이 표시되며, 워크플로우 애셋은 다음과 같은 메뉴를 제공합니다.

  • 새 워크플로우 시작 : 워크플로우 애셋을 사용하여 새로운 워크플로우를 생성 합니다.
  • 삭제 : 워크플로우 애셋을 삭제 합니다. 삭제 권한이 있을 때에만 표시 됩니다.

리스트 뷰 More Actions#

리스트의 more actions 버튼은 항상 표시되며, 카드 뷰와 동일한 메뉴를 제공 합니다.

다중 선택 More Actions#

여러개의 항목을 선택해서 More Actions를 실행할 수 있는 기능을 제공 합니다. 워크플로우를 생성할 때에는 하나의 워크플로우 애셋만 사용 가능하기 때문에, 선택한 항목 개수에 따라 표시되는 액션 버튼이 달라집니다.

  • 카드 뷰 다중선택

    카드에 마우스 커서를 올리면 선택 체크박스가 표시됩니다. 한개만 선택 했을 때에는 삭제와 워크플로우 시작 버튼이 표시 됩니다.

    두개 이상 선택 했을 때에는 삭제 버튼만 표시 됩니다.

  • 리스트 뷰 다중선택

    리스트의 선택 체크박스는 항상 표시되며, 선택한 항목의 개수에 따라 표시되는 액션 버튼은 카드 뷰와 동일 합니다. 한개만 선택 했을 때에는 삭제와 워크플로우 시작 버튼이 표시 됩니다.

    두개 이상 선택 했을 때에는 삭제 버튼만 표시 됩니다.

워크플로우 시작#

워크플로우 애셋을 사용해 새로운 워크플로우를 생성합니다.

More Actions에서 워크플로우 시작 버튼을 선택하면 다음과 같은 프로젝트 선택 팝업이 표시 됩니다.

프로젝트를 선택하면 선택한 프로젝트에 새로운 워크플로우가 생성 됩니다. 새로 생성된 워크플로우에는 워크플로우 애셋에 저장되어 있던 구성요소가 복제 됩니다.
