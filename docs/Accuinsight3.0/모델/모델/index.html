Screen Definition#

  1. GNB
  2. Header
  3. 탐색영역
  4. 모델 목록

1. GNB#

대시보드 페이지의 GNB 메뉴 항목을 참고해주세요.

2. Header#

  1. Chevron - 이전 화면으로 이동합니다.
  2. View 권한이상 포함된 모든 프로젝트 수를 표시합니다.
info

서비스 관리자의 경우 프로젝트 권한과 상관 없이 모든 프로젝트의 수가 표시됩니다.

3. 탐색영역#

  1. 보기 모드 - 카드 보기 모드 또는 리스트 보기 모드로 전환됩니다.
  2. 생성자 필터 - 모델을 생성한 사용자로 검색합니다.
  3. 프로젝트 필터 - 전체 프로젝트, 자신이 오너인 프로젝트로 검색합니다.
  4. 레지스트리 등록 이력 필터 - 전체, 레지스트리 등록 이력 있음으로 검색합니다.
  5. 정렬 필터 - 최신 업데이트 순, 제목 오름차순, 제목 내림차순으로 정렬합니다.
  6. 검색창 - 입력된 검색어로 검색합니다.

4. 모델 목록#

  1. 북마크 설정/해제
info

북마크 설정 시 우선적으로 정렬되어 리스트 상위에 표시 됩니다.

  1. 모델 More Action 메뉴
    • AI Runtime 이동 - AI Runtime 서비스로 이동합니다.
    • 편집 - 선택한 모델의 워크플로우로 이동합니다.
  2. 페이지 설정
  3. 한 페이지에 표시할 항목 수 설정
