스케줄러 생성

워크플로우를 선택하여 스케줄러를 생성 합니다.

워크플로우 선택#

스케줄러를 등록할 워크플로우를 선택 합니다.

  1. 검색 - 워크플로우를 검색
  2. 스케줄러를 등록할 워크플로우 선택
  3. 페이지 선택
  4. 취소 - 워크플로우 선택을 취소하고 팝업을 닫음
  5. 등록 - 선택한 워크플로우로 스케줄러 등록 화면으로 이동

스케줄러 등록하기#

1. 헤더#

  1. Chevron - 이전 화면으로 이동

2. 기본 정보 설정#

  1. 스케줄러 이름 입력
  2. 실행 이력 보관주기 선택 - 1일, 1주일, 1개월, 3개월, 6개월, 1년 중 선택

3. 스케줄 설정#

3-1. 기본방식#

  1. 입력 방식 - 기본방식, Cron 표현식 중 선택
  2. 시작 일시 - 스케줄러 시작 일시 입력
  3. 반복 주기 - 분, 시간, 일, 요일, 개월 단위로 반복 주기 입력
  4. 종료 일시
    • 날짜 지정 - 지정된 날짜에 종료
    • 횟수 반복 - 지정된 횟수만큼 반복 후 종료

3-2. Cron 표현식#

  1. 입력 방식 - 기본방식, Cron 표현식 중 선택
  2. Cron 표현식 입력 - 최소 분단위로 설정 가능

4. 알림 설정#

  1. 스케줄 시작 시 - 스케줄 시작 시 알림 전송 설정/해제
  2. 스케줄 종료 시 - 스케줄 종료 시 알림 전송 설정/해제
  3. 알림 채널 - 스케줄 알림 채널 설정/해제
info

서비스 관리자가 Admin 콘솔에서 알림 관리에 등록한 내용이 알림 채널에 표시 됩니다.

5. 변수 추가#

5-1. 워크플로우 변수#

  1. 변수 선택 - 워크플로우 변수, 프로젝트 변수 중 선택
  2. key값 추가 - 변수 추가
  3. Key - 변수의 키 입력
  4. Value - 변수의 값 입력
  5. 삭제 - 변수 삭제

5-2. 프로젝트 변수#

  1. 변수 선택 - 워크플로우 변수, 프로젝트 변수 중 선택
  2. 프로젝트 변수 선택 - 변수 그룹 관리에서 등록한 변수 중 선택
info

프로젝트 변수는 선택만 가능하고 수정이 불가능 합니다.
수정이 필요할 경우 변수 그룹 관리에서 수정을 해야 합니다.

6. 취소/등록#

  1. 입력한 내용을 취소하고 이전 화면으로 이동
  2. 입력한 내용으로 스케줄러 등록
