스케줄러 목록

워크플로우 스케줄링에 필요한 기능들을 제공합니다.

1. 헤더#

  1. Chevron - 이전 화면으로 이동

2. 탐색#

  1. 보기 모드 - 리스트 보기 모드와 캘린더 모드에 대해서 토클 수행
  2. 활성 상태 필터 - 활성 상태에 대해서 Active, Inactive 로 검색 수행
  3. 검색창 - 입력된 검색어로 검색 수행
  4. 삭제 - 체크한 스케줄러 삭제
  5. 스케줄러 등록 - 워크플로우 선택 팝업 표시 후 스케줄러를 등록할 워크플로우를 선택하면 상세 정보 입력 화면으로 이동

3. 스케줄러 목록#

  1. 전체 선택 - 모든 스케줄러 항목을 선택/해제
  2. 항목 선택 - 스케줄러 항목을 선택/해제
  3. 활성상태 - 스케줄러를 Active/Inactive로 설정
  4. 스케줄러의 More Actions 메뉴
    • 편집 - 스케줄러 편집 화면으로 이동
    • 복제 - 스케줄러 복제 수행
    • 삭제 - 스케줄러 삭제 수행
  5. 페이지 설정
  6. 한 페이지에 표시할 항목 수 설정

4. 캘린더 모드#

  • 월별 표시 - 일 단위로 스케줄러 동작 현황을 표시
  • 일별 표시 - 시간 단위로 스케줄러 동작 현황을 표시
