상세 화면

프로젝트에 포함된 워크플로우, 모델, 리포트와 공유 범위를 프로젝트(나만보기)로 저장한 애셋을 관리하고 스케줄러를 설정할 수 있습니다.

1. 헤더#

  1. Project - 프로젝트 목록으로 이동
  2. 스케줄러 - 스케줄러 목록으로 이동
  3. 저장된 애셋 - 저장된 애셋 목록으로 이동
  4. 구성원 - 구성원 권한 편집 팝업 표시
  5. History - 프로젝트 수정 내역 팝업 표시
  6. More Actions
    • 기본정보 편집 : 프로젝트 수정 화면으로 이동
    • 변수 그룹 관리 : 변수 그룹 관리 화면으로 이동
    • 프로젝트 삭제 : 프로젝트 삭제 수행
info

권한에 따라 헤더 매뉴가 비활성화 되거나 표시되지 않는 항목이 있습니다.
프로젝트 삭제는 프로젝트 구성요소(워크플로우, 모델)를 먼저 삭제해야 삭제가 가능 합니다.

2. 탐색#

  1. 보기 모드 - 카드 보기 모드와 리스트 보기 모드에 대해서 토클 수행
  2. 모두 선택 - 워크플로우를 모두 선택
  3. Owner 필터 - 워크플로우를 생성한 사용자로 검색 수행
  4. 소팅 필터 - 최신 업데이트 순, 제목 오름차순, 제목 내림차순으로 소팅 수행
  5. 검색창 - 입력된 검색어로 검색 수행
  6. 삭제 - 1개 이상의 워크플로우가 선택되면 표시되며 선택된 워크플로우 삭제 수행

4. 워크플로우 목록#

  1. 프로젝트에 작성된 워크플로우 수 표시
  2. 워크플로우 목록을 열거나 닫음
  3. 워크플로우 생성 화면으로 이동
  4. 북마크 설정/해제
info

북마크 설정 시 우선적으로 정렬되어 리스트 상위에 표시 됩니다.

  1. 워크플로우 선택
  2. 워크플로우에 스케줄러, 리포트가 포함되어 있으면 아이콘 표시
  3. 워크플로우의 More Actions 메뉴
    • 편집 - 워크플로우 상세 화면으로 이동
    • 복제 - 워크플로우 복제 수행
    • 삭제 - 워크플로우 삭제 수행
info

편집 시 Editor 권한이 없을 경우 읽기 전용 모드로 진입 합니다.

5. 모델 목록#

  1. 프로젝트에 작성된 모델 수 표시
  2. 모델 목록을 열거나 닫음
  3. 북마크 설정/해제
info

북마크 설정 시 우선적으로 정렬되어 리스트 상위에 표시 됩니다.

  1. 모델 이름으로 AI Runtime 배포 표시 (회색일 경우 미 배포, 파란색일 경우 배포 됨)
  2. 모델의 More Actions 메뉴
    • AI Runtime 이동 - AI Runtime 서비스의 모델 배포 현황 페이지로 이동
    • 편집 - 모델이 작성되어 있는 워크플로우 상세 화면으로 이동
info

AI Runtime 이동 시 배포가 삭제되었을 경우 오류 메시지가 발생할 수 있습니다.
편집 시 Editor 권한이 없을 경우 읽기 전용 모드로 진입 합니다.

6. 리포트 목록#

  1. 프로젝트에 작성된 리포트 수 표시
  2. 리포트 목록을 열거나 닫음
  3. 리포트의 More Actions 메뉴
    • 편집 - 리포트 편집 화면으로 이동
info

편집 시 Editor 권한이 없을 경우 읽기 전용 모드로 진입 합니다.
