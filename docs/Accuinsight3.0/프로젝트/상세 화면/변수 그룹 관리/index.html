변수 그룹 관리

워크플로우 실행 시 사용하는 글로벌 변수를 관리하는 메뉴입니다.
생성한 변수 그룹은 스케줄 생성 화면의 프로젝트 변수를 선택하여 사용할 수 있으며 워크플로우 변수로 선언된 변수보다 우선순위가 높습니다.

변수 그룹 관리#

  1. Chevron - 이전 화면으로 이동
  2. 검색 - 변수 그룹 이름으로 검색 수행
  3. 변수 그룹 추가 - 변수 그룹 추가 화면으로 이동
  4. More Actions
    • 수정 : 변수 그룹 수정 화면으로 이동
    • 삭제 : 변수 그룹 삭제 수행
  5. 페이지 설정
  6. 한 페이지에 표시할 항목 수 설정

변수 그룹 추가#

  1. Chevron - 이전 화면으로 이동
  2. 변수 그룹 이름 입력 - 최대 50자
  3. 가져오기 - csv 파일로 작성된 변수 정보를 로드 할 수 있으며, 추가되는 변수 값의 행 수만큼 반복 횟수가 추가되고 사용은 사용으로 셋팅
  4. Key값 추가 - 변수를 추가
  5. 사용 - 전체 회차를 한번에 사용 또는 미사용으로 셋팅
  6. 변수 이름 - 영역 클릭 시 수정으로 활성화되고 다른 영역을 크릭하면 비활성화 수행
  7. 변수 More Actions
    • 변수 수정 : 변수 값 입력란을 활성화
    • 변수 삭제 : 변수 삭제 수행
  8. 회차 핸들 - 핸들을 드래그하여 다른 회차와 순서 변경 수행
  9. 회차 사용 - 해당 회차를 사용 또는 미사용으로 셋팅
  10. 회차 More Actions
    • 회차 복사 : 회차 내용을 복사하여 추가
    • 회차 삭제 : 회차 삭제 수행
  11. 변수 값 입력
  12. 회차 추가 - 회차를 추가 함
  13. 취소 - 입력된 내용을 취소하고 이전 화면으로 이동
  14. 저장하기 - 입력된 내용으로 변수 그룹을 저장

변수 그룹 수정#

  1. 내보내기 - 변수 정보를(변수 명, 변수 값) csv 파일로 다운로드
