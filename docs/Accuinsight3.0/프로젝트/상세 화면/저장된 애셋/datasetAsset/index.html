데이터셋 애셋

데이터셋 애셋 개요#

프로젝트에 저장된 데이터셋 애셋을 조회/삭제/새 워크플로우 시작을 위해 관리합니다.

데이터셋 애셋 화면#

  1. Chevron - 이전 화면으로 이동
  2. 보기 모드: 애셋 목록을 카드 보기 혹은 리스트 보기를 토글 수행
  3. 나만 보기: 나만 보기 애셋을 체크하여 공유 권한이 '나만 보기'로 생성한 애셋 검색
  4. 애셋 구성요소 필터: 전체/리포트/Sandbox/모델로 애셋을 검색
  5. 소팅 필터: 최신 업데이트 순/오래된 순으로 정렬
  6. 입력창: 검색어를 입력하여 애셋명/설명을 검색
  7. 보이기/감추기: 데이터셋 애셋 목록을 보이기/감추기 토글 수행
  8. More Action 메뉴
    • 새 워크플로우 시작
    • 삭제
  9. 페이지 설정
  10. 한 페이지에 표시할 항목 수 설정

보기 모드#

카드 뷰와 리스트 뷰를 선택할 수 있습니다.

  • 데이터셋 애셋 카드

  • 데이터셋 애셋 리스트

나만 보기 애셋#

필터 선택 시 애셋 등록 시 공유 권한을 '나만 보기'로 지정한 애셋만 표시합니다.

애셋 구성요소 필터#

선택한 구성요소를 포함한 애셋만 표시합니다. 전체를 제외한 다중 선택이 가능합니다.

  • 전체
  • 리포트
  • Sandbox
  • 모델

소팅 필터#

  • 최신 업데이트 순 : 가장 최근에 업데이트된 애셋 순으로 정렬 됩니다.
  • 오래된 순 : 업데이트가 오래된 애셋 순으로 정렬 됩니다.

입력창#

검색어를 입력 하면 애셋 명/설명을 대상으로 검색된 결과를 표시합니다.

More Actions#

  • 워크플로우 애셋 카드 More Action
    카드에 마우스 커서를 올리면 More Action 버튼이 표시되며, 워크플로우 애셋은 다음과 같은 메뉴를 제공합니다.

    • 새 워크플로우 시작 : 워크플로우 애셋을 사용하여 새로운 워크플로우를 생성 합니다.
    • 삭제 : 워크플로우 애셋을 삭제 합니다. 삭제 권한이 있을 때에만 표시 됩니다.

  • 워크플로우 애셋 리스트 More Action More Action 버튼은 항상 표시되며, 카드 뷰와 동일한 메뉴를 제공 합니다.

다중 선택 More Actions#

여러개의 항목을 선택해서 More Actions를 실행할 수 있는 기능을 제공 합니다.

  • 카드 다중 선택
  • 리스트 다중 선택

데이터셋 애셋 기능 소개#

새 워크플로우 시작#

선택한 애셋에 저장된 데이터셋으로 새 워크플로우를 생성합니다.

상단에 위치한 '새 워크플로우 시작'버튼을 선택하면 프로젝트 선택 팝업이 표시 됩니다.

데이터 애셋 복사를 진행할 프로젝트를 선택하고 등록 버튼을 눌러 진행합니다.

데이터 애셋 복사 완료 후 선택한 프로젝트에 importAsset노드가 추가된 워크플로우가 생성됩니다.
