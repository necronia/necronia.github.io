공지사항 관리

서비스 이용에 대한 안내 및 공지사항 관리를 수행합니다.

공지사항 관리 화면#

1. 공지사항 검색#

  • 카테고리
    • 선택한 카테고리에 해당하는 공지사항을 목록에 표시합니다.
      • 전체
      • 일반
      • 개발
      • 운영
      • 업데이트
  • 상태
    • 선택한 상태에 해당하는 공지사항을 목록에 표시합니다.
      • 전체
      • 등록 완료
      • 임시 저장
  • 기간
    • 작성일이 기간 내에 있는 공지사항을 목록에 표시합니다.
  • 검색창
    • 제목이나 내용에 입력한 키워드가 포함된 공지사항을 목록에 표시합니다.

2. 공지사항 목록#

  • 카테고리
    • 공지사항 분류
  • 제목
    • 공지사항 제목
  • 내용
    • 공지사항 내용
  • 상태
    • 공지사항 상태
  • 게시일
    • 공지사항을 게시하는 기간
  • 작성자
    • 공지사항 작성자
  • 작성일
    • 공지사항 작성일시
  • More Actions 버튼
    • 편집
    • 게시
    • 삭제

3. 공지 등록#

  • 새로운 공지사항을 등록합니다.
    • 제목
      • 공지사항 제목을 입력합니다.
      • 필수 항목
    • 내용
      • 공지사항 내용을 입력합니다.
      • 필수 항목
    • 공지 기간
      • 공지사항을 게시할 기간을 설정합니다.
      • 필수 항목
    • 카테고리
      • 공지사항 분류를 설정합니다.
      • 필수 항목
    • 상태
      • 공지사항의 상태를 설정합니다.

4. More Actions 버튼#

  • 편집
    • 공지사항 정보를 수정합니다. 수정 가능한 정보는 다음과 같습니다.
      • 제목
      • 내용
      • 공지 기간
      • 카테고리
      • 상태
  • 게시
    • 임시 저장 상태의 공지사항을 등록 완료 상태로 변경하여 공지사항을 게시합니다.
  • 삭제
    • 공지사항을 삭제합니다.

5. 공지사항 사용자 노출#

  • 공지사항을 등록하면 공지 기간동안 사용자 로그인 시 글로벌 네이게이션 바 하단 영역에 공지사항을 보여줍니다.
  1. 공지사항
    • 공지사항 제목, 등록일시를 보여줍니다. 마우스 클릭 시 공지사항 내용을 보여주는 팝업이 나타납니다.
  2. 다시 보지 않기
    • 체크 후 닫기 버튼을 클릭하면 이후 로그인 시에는 해당 공지사항을 보여주지 않습니다.
  3. 닫기 버튼
    • 공지사항을 닫습니다.
