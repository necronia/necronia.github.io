사용자 관리

사용자 등록/삭제 및 권한을 부여하고 계정 관리를 수행합니다.

사용자 관리 화면#

1. 공지사항 검색#

  • 권한
    • 선택한 권한에 해당하는 사용자를 목록에 표시합니다.
      • 전체
      • 시스템 관리자
      • 서비스 관리자
      • 사용자
  • 상태
    • 선택한 상태에 해당하는 사용자를 목록에 표시합니다.
      • 전체
      • Active
      • Blocked
      • Locked
      • Deleted
  • 기간
    • 등록일이 기간 내에 있는 사용자를 목록에 표시합니다.
  • 검색창
    • 사용자 ID에 입력한 키워드가 포함된 사용자를 목록에 표시합니다.

2. 사용자 목록#

  • ID
  • 이름
  • Email
  • 권한
    • 시스템 관리자
    • 서비스 관리자
    • 사용자
  • 실패 횟수
    • 로그인 실패 횟수를 보여줍니다.
  • 상태
    • Active
    • Blocked
    • Locked
    • Deleted
  • 최근 접속일
  • 등록일
  • More Actions 버튼
    • 편집
    • 비밀번호 초기화
    • Blocked
    • 삭제

3. 사용자 등록#

  • 새로운 사용자를 등록합니다.
    • 이름
      • 필수 항목
    • 아이디
      • 필수 항목
    • E-mail
      • 필수 항목
    • 패스워드
      • 필수 항목
    • 그룹
      • 사용자가 포함될 그룹을 선택(다중 선택)할 수 있습니다.
      • 필수 항목
    • 권한
      • 시스템 관리자, 서비스 관리자, 사용자 중에서 선택할 수 있습니다.
      • 필수 항목
    • 상태
      • Active, Blocked 중에서 선택할 수 있습니다.
      • 필수 항목

4. More Actions 버튼#

  • 편집
    • 사용자 정보를 수정합니다. 수정 가능한 정보는 다음과 같습니다.
      • 이름
      • E-mail
      • 패스워드
      • 그룹
      • 권한
      • 상태
  • 비밀번호 초기화
    • 사용자의 비밀번호를 초기화합니다.
  • Blocked
    • 사용자 상태를 Blocked로 변경합니다.
    • Active 상태인 사용자만 보여지는 메뉴입니다.
  • Active
    • 사용자 상태를 Actvie로 변경합니다.
    • Blocked 상태인 사용자만 보여지는 메뉴입니다.
  • 삭제
    • 사용자를 삭제합니다.

5. 사용자 상세 정보 조회#

  • 목록에 있는 사용자를 클릭하면 상세 정보 조회 화면으로 이동합니다.
    • 요약 : 사용자의 요약 정보를 조회할 수 있습니다.
    • 접속 이력 : 사용자의 접속 이력을 조회할 수 있습니다.
    • 조회 이력 : 사용자 계정 상세 정보 및 데이터셋 정보 조회 이력을 볼 수 있습니다.
