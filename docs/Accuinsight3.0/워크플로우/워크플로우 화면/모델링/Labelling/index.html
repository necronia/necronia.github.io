Labelling

Labelling 은 text데이터를 분석하여 자동으로 알맞은 label을 달아줍니다. 몇 가지 샘플 라벨을 달아주거나 관련 문장을 학습시키면 더 정확한 label을 달 수 있습니다.

라벨링 생성#

워크플로우 작업 선택에서 Labelling으로 바로가기를 선택한 후 '다음' 버튼을 클릭하여 워크플로우를 생성합니다.

데이터 가져오기#

가져올 데이터를 선택하여 추가하면 Wrangler 노드에서 데이터를 확인할 수 있습니다.
건너뛰기 혹은 완료를 클릭하면 라벨링 노드가 준비됩니다.

라벨링 추가#

  1. 라벨링 노드를 선택하면 학습 방법, 대상 Text Column을 선택할 수 있는 우측 패널이 나타납니다.
  2. 학습 방법을 선택합니다.
    • Zero Shot
    • Few Shot
  3. 대상 Text Column을 선택합니다.
  4. 추가 버튼

Zero Shot#

  1. 학습 방법으로 Zero Shot을 선택하여 추가하였을 때의 설정 화면입니다.

  2. 라벨 추가 버튼을 클릭한 후 입력창에 라벨명을 입력하고 엔터를 눌러 라벨을 추가합니다.

  3. 라벨 추가 후 실행 버튼을 클릭하여 라벨링 작업을 시작합니다.

Few Shot#

  1. 학습 방법으로 Few Shot을 선택하여 추가하였을 때의 설정 화면입니다. 샘플 데이터 업로드 버튼을 클릭합니다.

  2. Browse 버튼을 클릭하여 샘플 데이터를 업로드합니다. 샘플 데이터 양식.csv를 클릭하여 파일을 다운로드 할 수 있습니다.

  3. 샘플 데이터를 업로드하면 파일 데이터를 파싱하여 추가할 라벨을 보여줍니다. 등록 버튼을 클릭하여 라벨을 추가합니다.

  4. 실행 버튼을 클릭하여 라벨링을 시작합니다.

라벨링 진행#

라벨링을 시작하면 우측 패널에서 선택한 학습 방법, 대상 Text Column과 상태를 볼 수 있습니다.

라벨링 완료#

라벨링이 완료되면 상태가 성공으로 업데이트 되고, 모델 노드가 추가됩니다. 열기 버튼을 클릭하면 라벨링 결과를 조회할 수 있습니다.

라벨링 결과 조회#

라벨링 결과를 조회하는 화면입니다. 라벨을 추가하거나 삭제한 후 재적용 버튼을 클릭하여 라벨링을 다시 실행할 수 있습니다.

라벨링 노드 옵션#

라벨링 노드를 마우스 우클릭하면 Labelling 노드 옵션이 나타납니다.

  1. Labelling 설정하기
    • 라벨링 결과를 조회하고 라벨 정보를 수정하여 라벨링을 재설정 할 수 있습니다.
  2. 이름 바꾸기
    • 노드 이름을 변경할 수 있습니다.
  3. 즐겨찾기에 추가
    • 우측 패널의 즐겨찾기에 라벨링 노드를 추가합니다.
  4. 즐겨찾기 삭제
    • 우측 패널의 즐겨찾기에서 라벨링 노드를 삭제합니다.
  5. 삭제
    • 라벨링 노드를 삭제합니다.

모델 노드#

라벨링이 완료된 후 모델 노드가 생성됩니다.

  1. 모델 노드
    • 라벨링이 완료된 후 생성된 모델 노드입니다.
  2. 모델 노드 옵션
    • 모델 레지스트리 등록
      • 새 모델 등록
        • AI Runtime에 새 모델을 등록합니다.
        • 등록할 모델이름 입력하고 중복확인 후 확인을 클릭하여 새 모델을 등록합니다.
      • 새 버전으로 등록
        • AI Runtime에 배포된 모델이 있을 경우 해당 모델의 새 버전으로 모델을 등록할 수 있습니다.
        • 배포된 모델 중 하나를 선택한 후 확인을 클릭하여 새 버전으로 모델을 등록합니다.
    • 이름 바꾸기
      • 모델 노드의 이름을 변경합니다.
    • 삭제
      • 모델 노드를 삭제합니다.
  3. 에측용 Workflow 생성
    • 현재 워크플로우 이름에 Predict prefix를 붙여서 예측용 워크플로우를 생성합니다.

예측용 Workflow#

워크플로우 목록에서 예측용 워크플로우 선택하여 불러옵니다.

  1. 데이터 가져오기를 클릭하여 가져올 데이터를 선택합니다.

  2. 데이터 가져오기가 완료된 후 모델 노드를 선택하고 우측 패널에서 예측 버튼을 클릭합니다.

  3. 예측 완료 후 예측 결과 데이터 노드가 생성됩니다.
