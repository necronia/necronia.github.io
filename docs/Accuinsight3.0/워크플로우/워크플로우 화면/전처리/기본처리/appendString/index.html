Append String

문자열 앞뒤에 다른 값을 붙이는 노드입니다.

워크플로우 보드에서 데이터 노드를 선택하고 노드 메뉴에서 [전처리] 카테고리의 [기본처리]노드중 [Append String]노드를 선택후 우측 패널에서 항목을 입력합니다.

워크플로우 보드에서 노드를 우클릭하면 메뉴항목을 볼 수 있습니다.

우측 패널#

  1. col : 접두어/접미어를 추가할 기준컬럼 선택. 대상 컬럼이 여러 개일 경우 좌측 [+]버튼 선택
  2. prefixString : 선택한 컬럼에 추가할 접두어 입력
  3. suffixString : 선택한 컬럼에 추가할 접미어 입력
  4. 추가 : 입력된 값을 적용하여 데이터 노드를 생성

메뉴#

  • 이름바꾸기 : 작성 중인 노드명 변경
  • 즐겨찾기에 추가 : 즐겨찾기에 [Append String]노드 추가
  • 삭제 : 해당 노드를 워크플로우에서 제거

예제#

타이타닉 데이터에서 Pclass 컬럼 앞에 '* ', 뒤에 '등급'자를 추가합니다.

  1. 워크플로우에 [타이타닉 데이터]를 로드

  2. [타이타닉 데이터] 노드에 [Cast] 노드를 연결하여 데이터셋을 생성

    • 적용할 Pclass 컬럼을 선택
    • dataType을 String으로 변경
    • 추가

  3. [생성된 데이터셋] 노드에 [Append String]노드를 연결하여 데이터셋을 생성

    • 접두어/접미어 적용대상 컬럼 Pclass를 선택
    • prefixString 값에 '* ' 입력
    • suffixString 값에 '등급' 입력
    • 추가
