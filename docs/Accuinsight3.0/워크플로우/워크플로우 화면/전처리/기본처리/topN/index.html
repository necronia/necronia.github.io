Top N

상위 몇 %의 데이터를 추출합니다.

워크플로우 보드에서 데이터 노드를 선택하고 노드 메뉴에서 [전처리] 카테고리의 [기본처리]노드중 [Top N]노드를 선택후 우측 패널에서 항목을 입력합니다.

워크플로우 보드에서 노드를 우클릭하면 메뉴항목을 볼 수 있습니다.

우측 패널#

  1. 적용할 column 선택 : 기준 컬럼 선택
  2. 상위 % : 상위 몇 프로 데이터를 추출할지 선택(1-100)
  3. 정렬 기준 : asc(오름차순), desc(내림차순)
  4. 추가 : 입력된 값을 적용하여 데이터 노드를 생성

메뉴#

  • 이름바꾸기 : 작성 중인 노드명 변경
  • 즐겨찾기에 추가 : 즐겨찾기에 [Top N]노드 추가
  • 삭제 : 해당 노드를 워크플로우에서 제거

예제#

타이타닉 데이터의 Age컬럼에서 상위 20%인 데이터만 불러오는 워크플로우를 생성합니다.

  1. 워크플로우에 [타이타닉 데이터]를 로드

  2. [타이타닉 데이터] 노드에 [Top N] 노드를 연결하여 데이터셋을 생성

    • 적용할 column 선택에 Age 컬럼을 선택
    • 상위 %에 20 입력
    • 정렬 기준에 desc 선택
    • 추가
