Select

필요 컬럼만 선택하여 조회합니다. 컬럼 선택 순서에 따라 노출 순서가 적용됩니다.

워크플로우 보드에서 데이터 노드를 선택하고 노드 메뉴에서 [전처리] 카테고리의 [기본처리]노드중 [Select]노드를 선택후 우측 패널에서 항목을 입력합니다.

워크플로우 보드에서 노드를 우클릭하면 메뉴항목을 볼 수 있습니다.

우측 패널#

  1. col : 필요한 컬럼 선택
  2. 추가 : 입력된 값을 적용하여 데이터 노드를 생성

메뉴#

  • 이름바꾸기 : 작성 중인 노드명 변경
  • 즐겨찾기에 추가 : 즐겨찾기에 [Select]노드 추가
  • 삭제 : 해당 노드를 워크플로우에서 제거

예제#

타이타닉 데이터에서 name, sex, age 값만 선택하는 워크플로우를 작성 합니다.

  1. 워크플로우에 [타이타닉 데이터]를 로드

  2. [타이타닉 데이터] 노드에 [Select]노드를 연결하여 데이터셋을 생성

    • 적용할 컬럼에 name, sex, age 컬럼을 선택
    • 추가
