Remove Trun Around

정해진 기간만큼의 데이터 행을 삭제합니다.

워크플로우 보드에서 데이터 노드를 선택하고 노드 메뉴에서 [전처리] 카테고리의 [기본처리]노드중 [Remove Trun Around]노드를 선택후 우측 패널에서 항목을 입력합니다.

워크플로우 보드에서 노드를 우클릭하면 메뉴항목을 볼 수 있습니다.

우측 패널#

  1. col : 날짜 컬럼을 선택
  2. dateFormat : 날짜 포맷을 지정
  3. start : 시작 날짜를 선택
  4. end : 종료 날짜를 선택
  5. extraNum : 날짜 범위 전후로 추가로 확인할 값
  6. extraType : 날짜 범위 전후로 추가로 확인할 값의 단위. 선택할 수 있는 extraType은 아래와 같다.
    • second
    • minute
    • hour
    • day
  7. 추가 : 입력된 값을 적용하여 데이터 노드를 생성

메뉴#

  • 이름바꾸기 : 작성 중인 노드명 변경
  • 즐겨찾기에 추가 : 즐겨찾기에 [Remove Trun Around]노드 추가
  • 삭제 : 해당 노드를 워크플로우에서 제거

예제#

timestamp 타입의 컬럼 정보가 특정 시작일자 ~ 종료일자에 해당하는 데이터만 남기고 삭제하는 워크플로우를 구성합니다.

  1. 워크플로우에 [shoping_web 데이터]를 로드

  2. [shoping_web 데이터] 노드에 [Remove Trun Around] 노드를 연결하여 데이터셋을 생성

    • col에 'http_date' 컬럼 선택
    • dateFormat에 'yyyy-MM-dd HH:mm:ss' 입력
    • start에 '2020-08-18 00:00' 선택
    • end에 '2020-08-19 00:00' 선택
    • 추가
