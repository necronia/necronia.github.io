Fill N/A

결측값을 지정한 문자로 치환합니다.

워크플로우 보드에서 데이터 노드를 선택하고 노드 메뉴에서 [전처리] 카테고리의 [기본처리]노드중 [Fill N/A]노드를 선택후 우측 패널에서 항목을 입력합니다.

워크플로우 보드에서 노드를 우클릭하면 메뉴항목을 볼 수 있습니다.

우측 패널#

  1. value : null값을 대체할 숫자 또는 문자 입력(문자열 입력시 쌍따옴표("") 사용필요)
  2. subset : 결측값 확인을 위한 컬럼 선택. 적용대상 컬럼이 여러 개일 경우 우측 [+]버튼으로 추가
    • col : 적용할 컬럼 선택
  3. 추가 : 입력된 값을 적용하여 데이터 노드를 생성

메뉴#

  • 이름바꾸기 : 작성 중인 노드명 변경
  • 즐겨찾기에 추가 : 즐겨찾기에 [Fill N/A]노드 추가
  • 삭제 : 해당 노드를 워크플로우에서 제거

예제#

타이타닉 데이터에서 Cabin 컬럼의 null 값을 "기타"로 변경합니다.

  1. 워크플로우에 [타이타닉 데이터]를 로드

  2. [타이타닉 데이터] 노드에 [Fill N/A]노드를 연결하여 데이터셋을 생성

    • value 값에 "기타" 입력
    • [더 보기]버튼을 클릭하여 입력 항목을 오픈
    • subset의 [+]버튼으로 입력 항목을 추가
    • 결측값을 대체할 Cabin 컬럼을 선택
    • 추가
