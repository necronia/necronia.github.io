Filter

원하는 데이터만 불러오기 위해 필터링 조건을 설정합니다.

워크플로우 보드에서 데이터 노드를 선택하고 노드 메뉴에서 [전처리] 카테고리의 [기본처리]노드중 [Filter]노드를 선택후 우측 패널에서 항목을 입력합니다.

우측 패널의 [더 보기] 버튼을 누르면 입력가능한 전체 항목을 볼 수 있습니다.

워크플로우 보드에서 노드를 우클릭하면 메뉴항목을 볼 수 있습니다.

우측 패널#

  1. filters : filtering 옵션 선택 적용대상 컬럼이 여러 개일 경우 우측 [+]버튼으로 추가
    • col : 필터링 대상 컬럼 선택
    • filterOption : 필터 조건 선택
    • filterValue : 필터 기준값 입력(문자열 입력시 쌍따옴표("") 사용필요)
  2. filterOperator : 필터 사이의 조건 연산자 선택(AND, OR)
  3. useCustom : 사용자 입력 기능 사용 시 선택
  4. customCondition : 사용자가 원하는 조건식을 직접 입력(1.filters로 해결할 수 없는 복잡한 조건 적용시)
  5. 추가 : 입력된 값을 적용하여 데이터 노드를 생성

메뉴#

  • 이름바꾸기 : 작성 중인 노드명 변경
  • 즐겨찾기에 추가 : 즐겨찾기에 [Filter]노드 추가
  • 삭제 : 해당 노드를 워크플로우에서 제거

예제#

타이타닉 데이터에서 Age컬럼이 60 이상이고, Pclass컬럼이 1인 데이터를 조회하는 워크플로우 생성

  1. 워크플로우에 [타이타닉 데이터]를 로드

  2. [타이타닉 데이터] 노드에 [Filter] 노드를 연결하여 데이터셋을 생성

    • 적용할 column 선택에 Age 컬럼을 선택
    • filters 의 [+] 버튼을 클릭하여 입력창을 하나 추가
    • 첫번째 col을 Age컬럼 선택
    • 첫번째 filterOption을 '> : greater than' 선택
    • 첫번째 filterValue에 60 입력
    • 두번째 col을 Pclass컬럼 선택
    • 두번째 filterOption을 '= : equal' 선택
    • 두번째 filterValue에 1 입력
    • [더 보기]버튼을 클릭하여 입력 항목을 오픈
    • filterOperator를 'AND' 선택
    • 추가
