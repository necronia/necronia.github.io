Remove Trun Around Auto

Trun Around를 기간 단위 기준으로 판단하여 해당 데이터 행을 삭제합니다. 예를 들어 특정 일자의 컬럼들 중 70% 이상이 0인것이 존재하면 해당 일자의 Row들은 전부 삭제합니다.

워크플로우 보드에서 데이터 노드를 선택하고 노드 메뉴에서 [전처리] 카테고리의 [기본처리]노드중 [Remove Trun Around Auto]노드를 선택후 우측 패널에서 항목을 입력합니다.

워크플로우 보드에서 노드를 우클릭하면 메뉴항목을 볼 수 있습니다.

우측 패널#

  1. col : 날짜 컬럼을 선택
  2. dateFormat : 날짜 포맷을 지정
  3. dateUnit : TA를 확인할 시간 단위
  4. ratio : TA의 비율
  5. 추가 : 입력된 값을 적용하여 데이터 노드를 생성

메뉴#

  • 이름바꾸기 : 작성 중인 노드명 변경
  • 즐겨찾기에 추가 : 즐겨찾기에 [Remove Trun Around Auto]노드 추가
  • 삭제 : 해당 노드를 워크플로우에서 제거

예제#

워크플로우를 구성합니다.

  1. 워크플로우에 [shoping_web 데이터]를 로드

  2. [shoping_web 데이터] 노드에 [Remove Trun Around Auto] 노드를 연결하여 데이터셋을 생성

    • col에 'http_date' 컬럼 선택
    • dateFormat에 'yyyy-MM-dd HH:mm:ss' 입력
    • dateUnit에 'day' 선택
    • ratio에 20 입력
    • 추가
