Cast

데이터 타입을 다른 타입으로 변환하는 노드입니다.

워크플로우 보드에서 데이터 노드를 선택하고 노드 메뉴에서 [전처리] 카테고리의 [기본처리]노드중 [Cast]노드를 선택후 우측 패널에서 항목을 입력합니다.

워크플로우 보드에서 노드를 우클릭하면 메뉴항목을 볼 수 있습니다.

우측 패널#

  1. subset : 변경대상 컬럼 및 데이터 타입 선택. 대상 컬럼이 여러 개일 경우 좌측 [+]버튼 선택
    • col : 변경대상 컬럼
    • dataType : 변경하고자 하는 데이터 타입
  2. 추가 : 입력된 값을 적용하여 데이터 노드를 생성

메뉴#

  • 이름바꾸기 : 작성 중인 노드명 변경
  • 즐겨찾기에 추가 : 즐겨찾기에 [Cast]노드 추가
  • 삭제 : 해당 노드를 워크플로우에서 제거

예제#

타이타닉 데이터 중 Age컬럼의 데이터 타입을 Integet로 변경합니다.

  1. 워크플로우에 [타이타닉 데이터]를 로드

  2. [타이타닉 데이터] 노드에 [Cast] 노드를 연결하여 데이터셋을 생성

    • 적용할 Age 컬럼을 선택
    • dataType을 Interger로 변경
    • 추가
