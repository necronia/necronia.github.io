IsIn

특정 값 포함 여부를 확인하여 그 결과를 True/False로 반환합니다.

워크플로우 보드에서 데이터 노드를 선택하고 노드 메뉴에서 [전처리] 카테고리의 [기본처리]노드중 [IsIn]노드를 선택후 우측 패널에서 항목을 입력합니다.

워크플로우 보드에서 노드를 우클릭하면 메뉴항목을 볼 수 있습니다.

우측 패널#

  1. col : 기준 컬럼 선택
  2. argument : 찾을 문자열 입력. 적용대상 문자열이 여러 개일 경우 우측 [+]버튼으로 추가 가능
  3. newColumn : True/False 결과가 입력될 새로운 컬럼명 입력
  4. 추가 : 입력된 값을 적용하여 데이터 노드를 생성

메뉴#

  • 이름바꾸기 : 작성 중인 노드명 변경
  • 즐겨찾기에 추가 : 즐겨찾기에 [IsIn]노드 추가
  • 삭제 : 해당 노드를 워크플로우에서 제거

argument 재사용#

[isin]노드에서 입력하는 argument를 저장하여 재사용할 수 있습니다. 문자열 입력 후 저장버튼을 누르면 argument가 저장됩니다.

  1. 저장 : 입력된 argument 저장 팝업 호출
  2. 목록 : 저장된 argument 리스트 팝업 호출
  3. 항목추가 : 적용대상 문자열을 추가
  4. 설명 : 저장할 설명 입력
  5. 확인 : argument 저장
  6. 적용 : 선택된 argument를 우측패널의 argument로 불러오기
  7. 삭제 : 저장된 argument를 삭제

예제#

타이타닉 데이터에서 Sex 컬럼에서 'male' 키워드를 찾아 man 컬럼을 만들어 True/False로 저장합니다.

  1. 워크플로우에 [타이타닉 데이터]를 로드

  2. [타이타닉 데이터] 노드에 [Select]노드를 연결하여 데이터셋을 생성

    • 적용할 컬럼에 PassengerId, Name, Sex, Age 컬럼을 선택
    • 추가

  3. [생성된 데이터셋] 노드에 [IsIn]노드를 연결하여 데이터셋을 생성

    • 대상 컬럼에 Sex 컬럼을 선택
    • argument 값입력
      • arg : 'male' 입력
    • newColumn 값에 'man' 입력
    • 추가
