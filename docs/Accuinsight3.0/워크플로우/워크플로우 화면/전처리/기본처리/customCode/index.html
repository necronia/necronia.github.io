Custom Code

PySpark의 DataFrame 함수를 사용하여 사용자 정의 코드를 입력 할 수 있다.

워크플로우 보드에서 데이터 노드를 선택하고 노드 메뉴에서 [전처리] 카테고리의 [기본처리]노드중 [Custom Code]노드를 선택후 우측 패널에서 항목을 입력합니다.

우측 패널의 [더 보기] 버튼을 누르면 입력가능한 전체 항목을 볼 수 있습니다.

워크플로우 보드에서 노드를 우클릭하면 메뉴항목을 볼 수 있습니다.

우측 패널#

  1. code : Dataframe에서 사용 가능한 함수가 포함된 코드(filter(), drop(), limit() 등) 작성
  2. variableType : 변수타입 입력 (spark DF, pandas DF, RDD)
  3. overwriteSchema : 실행결과에 대한 스키마를 재정의(체크하지 않을 경우 이전 스키마를 그대로 사용함)
  4. newSchema : overwriteSchema체크시 자동 입력
  5. 코드입력 팝업 : code 입력을 팝업으로 큰 화면에서 입력
  6. 추가 : 입력된 값을 적용하여 데이터 노드를 생성

메뉴#

  • 이름바꾸기 : 작성 중인 노드명 변경
  • 즐겨찾기에 추가 : 즐겨찾기에 [Custom Code]노드 추가
  • 삭제 : 해당 노드를 워크플로우에서 제거

Note#

dataset 변경 가능 (1줄 입력만 가능)ex. withColumn('temp_filled_spark', filled_column)

예제#

PySpark의 limit() 함수를 이용하여 3개 데이터만 표시하는 워크플로우를 구성합니다.

  1. 워크플로우에 [shoping_web 데이터]를 로드

  2. [shoping_web 데이터] 노드에 [Custom Code] 노드를 연결하여 데이터셋을 생성

    • code 'outputDf = inputDf.limit(3)' 입력
    • variableType에 'spark DF' 선택
    • 추가
