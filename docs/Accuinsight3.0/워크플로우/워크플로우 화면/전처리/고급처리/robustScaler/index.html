Robust Scaler

Definition#

Robust 스케일링 (median이 0, IQR(interquatile range)이 1이 되도록 변환) 하는 노드입니다. 우측 Components 패널에서 [전처리] - [고급처리]에서 [Robust Scaler]노드를 선택한 후 항목별 필요정보를 입력합니다.

Property#

  1. col : 대상 컬럼들 선택(숫자형 컬럼만 지원)
  2. withCentering : Median값을 중앙으로 데이터셋 정렬 여부 선택
  3. withScaling : IQR값으로 데이터셋 스케일링 여부 선택

Example#

  1. Dataset노드를 선택 후 우측 Components 패널에서 [전처리] - [고급처리]에서 노드를 선택하여 워크플로우에 생성
  1. 적용할 대상 컬럼(col)을 선택 후 실행
