열 나누기

Definition#

정의한 조건에 따라 대상 컬럼의 데이터내 구분자를 기준으로 단일 컬럼을 하나 이상의 컬럼으로 분할하여 신규 컬럼을 추가합니다.

info

열 나누기는 대상 컬럼의 데이터는 유지되며, 분할된 문자열의 개수만큼 컬럼이 추가됩니다.

Property#

  1. column : 대상 컬럼 선택
  2. delimiter : 구분자
  3. newColumns : 분할된 데이터가 포함될 컬럼명 설정(순서있음)

Example#

[Name] 컬럼의 데이터를 구분자(,) 기준으로 문자열을 [FirstName], [LastName]으로 분리하여 새로운 컬럼을 추가합니다.
