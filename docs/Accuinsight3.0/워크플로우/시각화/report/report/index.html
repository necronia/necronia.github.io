리포트 개요

리포트 생성시 구성 요소에 대하여 설명합니다.

리포트 생성#

차트 생성 화면의 "리포트 생성" 메뉴를 통해 리포트 생성 화면으로 진압할 수 있습니다.

  1. 리포트 생성 화면으로 진입할 수 있습니다.

리포트 생성 화면#

차트 생성 초기 화면 구성요소 입니다.

  1. 리포트의 이름을 변경할 수 있습니다.
  2. 현재 구성된 리포트를 초기화 합니다.
  3. 리포트를 삭제합니다.
  4. 리포트를 저장합니다.
  5. 리포트를 다운로드 합니다.
    • 이미지 형식의 다운로드를 지원합니다.
  6. 작성중인 리포트의 미리보기를 제공합니다.
    • 미리보기 화면에서도 5번의 리포트 다운로드 기능을 제공합니다.
  7. 새로운 리포트를 추가합니다.
  8. 리포트가 표출되는 영역입니다.
  9. 차트를 검색할 수 있습니다. (대, 소문자 구분)
  10. 저장된 차트 목록을 표출합니다.
  11. 리포트에 추가할 수 있는 오브젝트 목록입니다. (아래 4가지의 오브젝트를 제공합니다.)
    • 텍스트
    • 이미지
    • HTTP 링크
    • 빈 오브젝트
  12. 리포트의 배경색을 변경합니다.
  13. 배경색의 투명도를 조절합니다.
  14. 리포트의 테두리색을 변경합니다.
  15. 리포트의 테두리색 투명도를 조절합니다.
  16. 리포트의 테두리 두께를 조절합니다.
