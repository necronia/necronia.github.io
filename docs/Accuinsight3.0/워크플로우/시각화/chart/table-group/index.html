Table-group chart

Table-group chart를 생성하는 방법에 대하여 설명합니다.

차트 생성 초기 화면#

Table-group chart 생성 초기 화면입니다.

  1. 선택된 데이터셋에 포함된 컬럼을 표출합니다.
  2. 컬럼을 검색할 수 있습니다. (대,소문자 구분)
  3. 그룹의 값을 선택할 수 없습니다.
  4. 값을 선택합니다.
    • 1개의 컬럼 선택이 가능합니다.
  5. 집계함수를 선택합니다. (아래 6개의 집계 함수를 제공합니다.)
    • MIN
    • MAX
    • COUNT
    • SUM
    • AVG
    • DISTINCT

선택된 컬럼 관계#

Table-group chart 생성시 선택된 컬럼에 대한 처리 방법입니다.

info

그룹, 컬럼, 집계홤수 관계:

  • 그룹은 SQL 쿼리의 group by 구문으로 적용됩니다.
  • 집계홤수는 컬럼에 선택된 값에 적용됩니다.
