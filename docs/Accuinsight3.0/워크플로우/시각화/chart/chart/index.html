차트 생성시 구성 요소에 대하여 설명합니다.

차트 생성#

workflow 노드를 선택 후 마우스 오른쪽 클릭을 통해 차트 생성 화면으로 진압할 수 있습니다.

차트 생성 화면#

차트 생성 초기 화면 구성요소 입니다.

  1. 차트 종류를 선택할 수 있습니다.
  2. 현재 구성된 차트를 초기화 합니다.
  3. 차트를 삭제합니다.
  4. 차트를 저장합니다.
  5. 저장된 차트를 다운로드 합니다.
  6. 생성된 차트를 이용하여 리포트를 작성합니다.
  7. 차트 제목을 변경합니다.
  8. 새로운 차트를 추가합니다.
  9. 차트가 표출되는 영역입니다.
