범례

범례 옵션 종류입니다.

  1. Orient
    • 범례의 레이아웃 방향을 지정합니다.
    • 아래 2가지 값 선택이 가능합니다.
      • horizontal
      • vertical
  2. Sub Text Style
    1. Color
      • 글씨 색상을 설정합니다.
      • 입력형식: rgba 형식으로 지정됩니다. (예: #FF0000FF)
    2. Font Style
      • 글씨 스타일을 지정합니다.
      • 아래 3가지 값 선택이 가능합니다.
        • normal
        • italic
        • oblique
    3. Font Weight
      • 글씨 두께를 설정합니다.
      • 입력형식: 문자 또는 숫자 형식으로 지정됩니다.
        • 입력 가능한 문자는 아래와 같습니다.
          • normal
          • bold
          • bolder
          • lighter
        • 입력 가능한 숫자는 아래와 같습니다.
          • 100, 200, 300, 400 등
    4. Font Size
      • 글씨 크기를 설정합니다.
      • 입력형식: 숫자 형식으로 지정됩니다. (예: 12)
    5. Align
      • 글씨 스타일을 지정합니다.
      • 아래 3가지 값 선택이 가능합니다.
        • left
        • center
        • right
  3. bottom
    • 범례와 바닥과의 거리를 지정합니다.
    • 입력형식: 문자 또는 숫자 형식으로 지정됩니다.
      • 입력 가능한 문자는 아래와 같습니다.
        • auto
      • 입력 가능한 숫자 형식은 아래와 같습니다.
        • 20 : pixel 단위로 거리를 지정합니다.
        • 20% : 백분율 단위로 거리를 지정합니다.

  1. Item Height

    • 범례 심볼의 이미지 높이를 지정합니다.
    • 입력형식: 숫자 형식으로 지정됩니다.

  2. Show

    • 범례 표출 여부를 지정하니다.

  3. Icon

    • 범례 항목의 아이콘을 지정합니다.
    • 아래 가지 값 선택이 가능합니다.
      • circle
      • rect
      • roundRect
      • triangle
      • diamond
      • pin
      • arrow
      • none

  4. Right

    • 범례의 오른쪽 거리를 지정합니다.
    • 입력형식: 문자 또는 숫자 형식으로 지정됩니다.
      • 입력 가능한 문자는 아래와 같습니다.
        • auto
      • 입력 가능한 숫자 형식은 아래와 같습니다.
        • 20 : pixel 단위로 거리를 지정합니다.
        • 20% : 백분율 단위로 거리를 지정합니다.

  5. Align

    • 범례 마크와 범례 텍스트이 정렬을 지정합니다.
    • 아래 3가지 값 선택이 가능합니다.
      • auto
      • left
      • right

  6. Top

    • 범례 위쪽의 거리를 지정합니다.
    • 입력형식: 문자 또는 숫자 형식으로 지정됩니다.
      • 입력 가능한 문자는 아래와 같습니다.
        • auto
        • top
        • middle
        • bottom
      • 입력 가능한 숫자 형식은 아래와 같습니다.
        • 20 : pixel 단위로 거리를 지정합니다.
        • 20% : 백분율 단위로 거리를 지정합니다.

  7. Left

    • 범례 왼쪽의 거리를 지정합니다. 단
          - 20% : 백분율 단위로 거리를 지정합니다.

  8. Formatter

    • 범례의 레이블 형식을 지정하는 데 사용됩니다.
    범례 형식 지정 예시
        formatter: 'Legend {name}'

  1. Selected Mode

    • 범례를 클릭하여 해당 범례의 시리즈 표시 여부를 제어합니다.

  2. Width

    • 범례의 폭을 지정합니다.
    • 입력형식: 문자 또는 숫자 형식으로 지정됩니다.
      • 입력 가능한 문자는 아래와 같습니다.
        • auto
      • 입력 가능한 숫자 형식은 아래와 같습니다.
        • 20 : pixel 단위로 거리를 지정합니다.
        • 20% : 백분율 단위로 거리를 지정합니다.

  3. Height

    • 범례의 높이를 지정합니다.
      • 입력형식: 문자 또는 숫자 형식으로 지정됩니다.
        • 입력 가능한 문자는 아래와 같습니다.
          • auto
        • 입력 가능한 숫자 형식은 아래와 같습니다.
          • 20 : pixel 단위로 거리를 지정합니다.
          • 20% : 백분율 단위로 거리를 지정합니다.
