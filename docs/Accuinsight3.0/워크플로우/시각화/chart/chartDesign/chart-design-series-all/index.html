시리즈

시리즈 옵션 종류입니다.

  • 시리즈 옵션은 차트 종류별로 차이가 있습니다.

공통 시리즈 옵션#

각 차트별 공통적인 시리즈 옵션에 대하여 설명합니다.

  1. Show Symbol

    • 심볼 표시 여부를 지정합니다.

  2. Symbol

    • 심볼의 아이콘 타입을 지정합니다.
    • 아래 8가지 값 선택이 가능합니다.
      • circle
      • rect
      • roundRect
      • triangle
      • diamond
      • pin
      • arrow
      • none

  3. Symbol Size

    • 심볼의 크기를 지정합니다.
    • 입력형식: 숫자 형식으로 지정됩니다. (예: 12)

  4. Stack

    • 스택 이름을 지정하면 동일한 범주 축에서 동일한 스택 이름을 가진 시리즈가 맨 위에 놓입니다.
    • 입력형식: 문자 형식으로 지정됩니다.

  5. Step

    • 스텝 라인을 표출할지 여부를 지정합니다.

  1. Label

    • 시리즈의 데이터 값등을 표시할지 여부를 지정합니다.
    1. Show
      • 라벨 표출 여부를 지정합니다.
    2. Position
      • 아래 13가지 값 선택이 가능합니다.
        • top
        • left
        • right
        • bottom
        • inside
        • insideLeft
        • insideRight
        • insideTop
        • insideBottom
        • insideTopLeft
        • insideBottomLeft
        • insideTopRight
        • insideBottomRight
    3. Rotate
      • 레이블을 -90도에서 90도까지 회전합니다.
        • 양수 값은 시계 반대 방향으로 회전을 나타냅니다.
    4. Color
      • 라벨 색상을 설정합니다.
      • 입력형식: rgba 형식으로 지정됩니다. (예: #FF0000FF)
    5. Font Style
      • 글씨 스타일을 지정합니다.
      • 아래 3가지 값 선택이 가능합니다.
        • normal
        • italic
        • oblique
    6. Font Weight
      • 글씨 두께를 설정합니다.
      • 입력형식: 문자 또는 숫자 형식으로 지정됩니다.
        • 입력 가능한 문자는 아래와 같습니다.
          • normal
          • bold
          • bolder
          • lighter
        • 입력 가능한 숫자는 아래와 같습니다.
          • 100, 200, 300, 400 등
    7. Font Style
      • 글씨 스타일을 지정합니다.
      • 아래 3가지 값 선택이 가능합니다.
        • normal
        • italic
        • oblique
    8. Align
      • 글씨 스타일을 지정합니다.
      • 아래 3가지 값 선택이 가능합니다.
        • left
        • center
        • right

  2. Line Style

    1. Width
      • 라인의 두께를 지정합니다.
      • 입력형식: 문자 또는 숫자 형식으로 지정됩니다.
    2. Type
      • 라인의 종류를 지정합니다.
      • 아래 3가지 값 선택이 가능합니다.
        • solid
        • dashed
        • dotted

  3. Smooth

    • 라인을 부드러운 커브로 보여줄지를 지정합니다.

Line, Scatter 차트 시리즈 옵션#

  1. Mark Line
    • 예측 기준값을 지정한 경우에 활성화 됩니다.
    • 차트 공통 시리즈와 동일한 속성을 가집니다.

Area 차트 시리즈 옵션#

  1. Color
    • 영역의 색상을 설정합니다.
    • 입력형식: rgba 형식으로 지정됩니다. (예: #FF0000FF)
  2. Opacity
    • 영역의 투명도를 설정합니다.
    • 입력형식: 숫자 형식으로 지정됩니다.
      • 입력 가능한 범위는 0부터 1사이의 값입니다.

Bar, Bar-Line, Histogram 차트 시리즈 옵션#

  1. Background Style

    1. Color

      • 영역의 색상을 설정합니다.
      • 입력형식: rgba 형식으로 지정됩니다. (예: #FF0000FF)

    2. Opacity

      • 영역의 투명도를 설정합니다.
      • 입력형식: 숫자 형식으로 지정됩니다.
        • 입력 가능한 범위는 0부터 1사이의 값입니다.

  1. Bar Width

    • Bar의 폭을 지정합니다.
    • 입력형식: 문자 또는 숫자 형식으로 지정됩니다.
      • 입력 가능한 문자는 아래와 같습니다.
        • '20%' : 백분율 단위로 폭을 지정합니다.
      • 입력 가능한 숫자 형식은 아래와 같습니다.
        • 20 : 절대값 단위로 폭을 지정합니다.

  2. Bar Gap

    • 각 시리즈간의 간격을 지정합니다.
    • 입력형식: 문자 형식으로 지정됩니다.
      • 입력 가능한 문자는 아래와 같습니다.
        • '20%' : 백분율 단위로 간격을 지정합니다.

  3. Stack

    • ## 공통 시리즈 옵션의 Stack을 참조합니다.

  4. Item Style

    • Color
      • 바의 색상을 설정합니다.
      • 입력형식: rgba 형식으로 지정됩니다. (예: #FF0000FF)
    • Border Color
      • 바의 테두리 색상을 설정합니다.
      • 입력형식: rgba 형식으로 지정됩니다. (예: #FF0000FF)
    • Border Width
      • 바의 테두리 두께를 설정합니다.
      • 입력형식: 숫자 형식으로 지정됩니다. (예: 12)
    • Border Type
      • 바의 테두리 유형 설정합니다.
      • 아래 3가지 값 선택이 가능합니다.
        • solid
        • dashed
        • dotted
    • 리
      • 바의 둥근 모서리의 반경입니다.
      • 입력형식: 숫자 형식으로 지정됩니다. (예: 12)
    • Opacity
      • 영역의 투명도를 설정합니다.
      • 입력형식: 숫자 형식으로 지정됩니다.
        • 입력 가능한 범위는 0부터 1사이의 값입니다.

Column, Column-Line, Column-Stacked 차트 시리즈 옵션#

  • ## 공통 시리즈 옵션을 참조합니다.

Pie, Donut 차트 시리즈 옵션#

  1. Clockwise

    • 원형 차트의 구역 레이아웃을 시계 방향으로 표시할지 여부를 지정합니.

  2. Start Angle

    • 시작 각도를 지정하니다.
    • 입력형식: 숫자 형식으로 지정됩니다.
      • 범위는 [0, 360] 입니다.

  3. Rose Type

    • 나이팅게일 차트 형식으로 표시할지 여부를 지정하니다.

  4. Left

    • 파이차트와 왼쪽의 거리를 지정합니다.
    • 입력형식: 문자 또는 숫자 형식으로 지정됩니다.
      • 입력 가능한 문자는 아래와 같습니다.
        • auto
        • left
        • center
        • right
      • 입력 가능한 숫자 형식은 아래와 같습니다.
        • 20 : pixel 단위로 거리를 지정합니다.
        • 20% : 백분율 단위로 거리를 지정합니다.

  5. Top

    • 파이차트와 위쪽의 거리를 지정합니다.
    • 입력형식: 문자 또는 숫자 형식으로 지정됩니다.
      • 입력 가능한 문자는 아래와 같습니다.
        • auto
        • top
        • middle
        • bottom
      • 입력 가능한 숫자 형식은 아래와 같습니다.
        • 20 : pixel 단위로 거리를 지정합니다.
        • 20% : 백분율 단위로 거리를 지정합니다.

  6. Right

    • 파이차트와 오른쪽의 거리를 지정합니다.
    • 입력형식: 문자 또는 숫자 형식으로 지정됩니다.
      • 입력 가능한 문자는 아래와 같습니다.
        • '20%' : 백분율 단위로 거리를 지정합니다.
      • 입력 가능한 숫자 형식은 아래와 같습니다.
        • 20 : pixel 단위로 거리를 지정합니다.

  7. Bottom

    • 파이차트와 바닥의 거리를 지정합니다.
    • 입력형식: 문자 또는 숫자 형식으로 지정됩니다.
      • 입력 가능한 문자는 아래와 같습니다.
        • '20%' : 백분율 단위로 거리를 지정합니다.
      • 입력 가능한 숫자 형식은 아래와 같습니다.
        • 20 : pixel 단위로 거리를 지정합니다.

  8. Width

    • 파이차트의 폭을 지정합니다.
    • 입력형식: 문자 또는 숫자 형식으로 지정됩니다.
      • 입력 가능한 문자는 아래와 같습니다.
        • '20%' : 백분율 단위로 폭을 지정합니다.
      • 입력 가능한 숫자 형식은 아래와 같습니다.
        • 20 : pixel 단위로 폭을 지정합니다.

  9. Height

    • 파이차트의 높이를 지정합니다.
      • 입력형식: 문자 또는 숫자 형식으로 지정됩니다.
        • 입력 가능한 문자는 아래와 같습니다.
          • '20%' : 백분율 단위로 높이를 지정합니다.
        • 입력 가능한 숫자 형식은 아래와 같습니다.
          • 20 : pixel 단위로 높이를 지정합니다.

  1. Label

    • 기본적으로 ## 공통 시리즈 옵션의 Label을 참조합니다.

      info

      공통 부분의 Label과 다른 부분을 설명합니다.

    • Position

      • 라벨의 위치를 지정합니다.
      • 아래 3가지 값 선택이 가능합니다.
        • outside
        • inside
        • center

    • Distance To Label Line

      • 라벨선과 텍스트와의 거리를 지정합니다.
      • 입력형식: 숫자 형식으로 지정됩니다. (예: 12)

    • Edge Distance

      • 텍스트 끝과 뷰포트 사이의 수평 거리를 지정합니다.
      • 입력형식: 문자 또는 숫자 형식으로 지정됩니다.
        - 입력 가능한 문자는 아래와 같습니다.    - '20%' : 백분율 단위로 수평 거리를 지정합니다.- 입력 가능한  숫자 형식은 아래와 같습니다.- 20 : 수평 거리를 지정합니다.

  1. Label Line

    • Show

      • 라벨 가이드 라인 표시 여부를 지정합니다.

    • Length

      • 가이드 라인의 길이를 지정합니다.
      • 입력형식: 숫자 형식으로 지정됩니다. (예: 12)

    • Smooth

      • 가이드 라인선을 부드럽게 표시합니다.
      • 입력형식: 숫자 형식으로 지정됩니다.
        • 입력 가능한 범위는 0부터 1사이의 값입니다.

  2. Item Style

    • Color

      • 차트의 색상을 설정합니다.
      • 입력형식: rgba 형식으로 지정됩니다. (예: #FF0000FF)

    • Border Color

      • 차트의 테두리 색상을 설정합니다.
      • 입력형식: rgba 형식으로 지정됩니다. (예: #FF0000FF)

    • Border Width

      • 차트의 테두리 두께를 설정합니다.
      • 입력형식: 숫자 형식으로 지정됩니다. (예: 12)

    • Border Type

      • 차트의 테두리 유형을 설정합니다.
      • 아래 3가지 값 선택이 가능합니다.
        • solid
        • dashed
        • dotted

    • Border Radius

      • 차트의 둥근 모서리의 반경입니다.
      • 입력형식: 숫자 형식으로 지정됩니다. (예: 12)

    • Opacity

      • 영역의 투명도를 설정합니다.
      • 입력형식: 숫자 형식으로 지정됩니다.
        • 입력 가능한 범위는 0부터 1사이의 값입니다.

  1. Label

    • Border Color
      • 바의 테두리 색상을 설정합니다.

  4. Z

    • Zlevel과 동일하나 우선 순위가 Zlevel 보다 낫습니다.

  5. Animation

    • 애미메이션 활성화 여부를 지정합니다.

Gauge 차트 시리즈 옵션#

  1. Start Angle

    • Gauge 차트의 시작 각도를 지정합니다.
    • 입력형식: 숫자 형식으로 지정됩니다.
      • 입력 가능한 범위는 0부터 360사이의 값입니다.

  2. End Angle

    • Gauge 차트의 끝 각도를 지정합니다.
    • 입력형식: 숫자 형식으로 지정됩니다.
      • 입력 가능한 범위는 0부터 360사이의 값입니다.

  3. Min

    • Gauge 차트의 최소값을 지정합니다.

  4. Max

    • Gauge 차트의 최대값을 지정합니다.

  5. Anchor

    • 앵커 표시 여부를 지정합니다.

  1. Pointer

    • Icon
      • 포인터의 타입을 지정합니다.
      • 아래 8가지 값 선택이 가능합니다.
        • circle
        • rect
        • roundRect
        • triangle
        • diamond
        • pin
        • arrow
        • none
    • Width
      • 포인터의 폭을 지정합니다.
      • 입력형식: 숫자 형식으로 지정됩니다. (예: 12)

  2. Progress

    • Show
      • Progress 표시 여부를 지정합니다.
    • Overlap
      • 데이터 그룹이 여러개 있을 경우 Progress 덮어씌우기 여부를 지정합니다.
    • RoundCap
      • Progress 끝을 둥글게 표시할지 여부를 지정합니다.

  3. Axis Line

    • Round Cap
      • 축의 끝을 둥글게 표시할지 여부를 지정합니다.

  4. Title

    • Font Size
      • 제목 글씨의 크기를 지정합니다.
      • 입력형식: 숫자 형식으로 지정됩니다. (예: 12)

  1. Detail
    • Height
      • Detail의 높이를 지정합니다.
      • 입력형식: 숫자 형식으로 지정됩니다. (예: 12)
    • Font Size
      • Detail의 글씨 크기를 지정합니다.
      • 입력형식: 숫자 형식으로 지정됩니다. (예: 12)
    • Background Color
      • 배경 색상을 설정합니다.
      • 입력형식: rgba 형식으로 지정됩니다. (예: #FF0000FF)
    • Border Radius
      • 둥근 모서리의 반경입니다.
      • 입력형식: 숫자 형식으로 지정됩니다. (예: 12)
    • Formatter
      • 툴팁의 Formatter 참조

Funnel 차트 시리즈 옵션#

  1. Min

    • Funnel 차트의 최소값을 지정합니다.

  2. Max

    • Funnel 차트의 최대값을 지정합니다.

  3. Min Size

    • 최소값이 표출될 폭을 지정합니다.
    • 입력형식: 문자 또는 숫자 형식으로 지정됩니다.
      • 입력 가능한 문자는 아래와 같습니다.
        • '20%' : 백분율 단위로 폭을 지정합니다.
      • 입력 가능한 숫자 형식은 아래와 같습니다.
        • 20 : 절대값 단위로 폭을 지정합니다.

  4. Max Size

    • 최대값이 표출될 폭을 지정합니다.
    • 입력형식: 문자 또는 숫자 형식으로 지정됩니다.
      • 입력 가능한 문자는 아래와 같습니다.
        • '20%' : 백분율 단위로 폭을 지정합니다.
      • 입력 가능한 숫자 형식은 아래와 같습니다.
        • 20 : 절대값 단위로 폭을 지정합니다.

  5. Orient

    • 차트의 방향을 지정합니다.
    • 아래 2가지 값 선택이 가능합니다.
      • vertical
      • horizontal

  6. Sort

    • 데이터 정렬 방식을 지정합니다.
    • 아래 3가지 값 선택이 가능합니다.
      • ascending
      • descending
      • none

  7. Gap

    • 사다리꼴 사이의 간격을 지정합니다.
    • 입력형식: 숫자 형식으로 지정됩니다. (예: 12)

  8. Legend Hover Link

    • 범례를 마우스로 호버할 경우 차트 강조 표시 활성화 여부를 지정합니다.

  9. Funnel Align

    • 수평 정렬 방식을 지정합니다.
    • 아래 3가지 값 선택이 가능합니다.
      • left
      • right
      • center

  1. Label

    • 기본적으로 ## 공통 시리즈 옵션의 Label을 참조합니다.

      info

      공통 부분의 Label과 다른 부분을 설명합니다.

    • Vertical Align

      • 텍스트의 수직정렬을 지정합니다.
        • 아래 3가지 값 선택이 가능합니다.
          • top
          • middle
          • bottom

  2. Label Line

    • Show
      • 가이드라인 표시여부를 지정합니다.
    • Length
      • 가이드라인의 길이를 지정합니다.
      • 입력형식: 숫자 형식으로 지정됩니다. (예: 12)

  3. Item Style

    • Color

      • 차트의 색상을 설정합니다.
      • 입력형식: rgba 형식으로 지정됩니다. (예: #FF0000FF)

    • Border Color

      • 차트의 테두리 색상을 설정합니다.
      • 입력형식: rgba 형식으로 지정됩니다. (예: #FF0000FF)

    • Opacity

      • 영역의 투명도를 설정합니다.
      • 입력형식: 숫자 형식으로 지정됩니다.
        • 입력 가능한 범위는 0부터 1사이의 값입니다.

  1. Left

    • 차트와 왼쪽의 거리를 지정합니다.
    • 입력형식: 문자 또는 숫자 형식으로 지정됩니다.
      • 입력 가능한 문자는 아래와 같습니다.
        • auto
        • left
        • center
        • right
      • 입력 가능한 숫자 형식은 아래와 같습니다.
        • 20 : pixel 단위로 거리를 지정합니다.
        • 20% : 백분율 단위로 거리를 지정합니다.

  2. Top

    • 차트와 위쪽의 거리를 지정합니다.
    • 입력형식: 문자 또는 숫자 형식으로 지정됩니다.
      • 입력 가능한 문자는 아래와 같습니다.
        • auto
        • top
        • middle
        • bottom
      • 입력 가능한 숫자 형식은 아래와 같습니다.
        • 20 : pixel 단위로 거리를 지정합니다.
        • 20% : 백분율 단위로 거리를 지정합니다.

  3. Right

    • 차트와 오른쪽의 거리를 지정합니다.
    • 입력형식: 문자 또는 숫자 형식으로 지정됩니다.
      • 입력 가능한 문자는 아래와 같습니다.
        • '20%' : 백분율 단위로 거리를 지정합니다.
      • 입력 가능한 숫자 형식은 아래와 같습니다.
        • 20 : pixel 단위로 거리를 지정합니다.

  4. Bottom

    • 차트와 바닥의 거리를 지정합니다.
    • 입력형식: 문자 또는 숫자 형식으로 지정됩니다.
      • 입력 가능한 문자는 아래와 같습니다.
        • '20%' : 백분율 단위로 거리를 지정합니다.
      • 입력 가능한 숫자 형식은 아래와 같습니다.
        • 20 : pixel 단위로 거리를 지정합니다.

  5. Width

    • 차트의 폭을 지정합니다.
    • 입력형식: 문자 또는 숫자 형식으로 지정됩니다.
      • 입력 가능한 문자는 아래와 같습니다.
        • '20%' : 백분율 단위로 폭을 지정합니다.
      • 입력 가능한 숫자 형식은 아래와 같습니다.
        • 20 : pixel 단위로 폭을 지정합니다.

  6. Height

    • 차트의 높이를 지정합니다.
      • 입력형식: 문자 또는 숫자 형식으로 지정됩니다.
        • 입력 가능한 문자는 아래와 같습니다.
          • '20%' : 백분율 단위로 높이를 지정합니다.
        • 입력 가능한 숫자 형식은 아래와 같습니다.
          • 20 : pixel 단위로 높이를 지정합니다.

  7. Series Layout By

    • 데이터셋의 column, row 중에서 Series에 대응되는 값을 지정합니다.
    • 아래 2가지 값 선택이 가능합니다.
      • column
      • row

  8. Dataset Index

    • 차트에 사용될 데이터셋의 인덱스를 지정합니다.
    • 입력형식: 숫자 형식으로 지정됩니다. (예: 12)
